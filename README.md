#HOW TO RUN IT?

 How to Run the Project
1. Create a virtual environment:
```bash
python -m venv venv
```
2. Activate the virtual environment:
```bash
venv\Scripts\activate
```
3. Install Groq library:
```bash
pip install groq
```
4. Add your Groq API key inside `chat.py`
5. Run the chatbot:
```bash
python chat.py
```
